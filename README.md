# watermark_generator
# Генератор водяных знаков

### Requirements

 - PHP 5.3+ with GD and mbstring

### Установка и запуск

- git clone https://github.com/T-34/watermark_generator
- cd watermark_generator
- bower i && npm i
- composer install
- gulp
